# 800m Athletics

Resursă completă pentru atleții de 800m - antrenamente, strategii, nutriție și noutăți.

## Despre Proiect

Acest site este dedicat probei de 800m din atletism, oferind informații valoroase pentru atleți de toate nivelurile:

- **Antrenamente specializate** pentru începători, intermediari și avansați
- **Strategii de cursă** pentru a maximiza performanța
- **Sfaturi de nutriție** pentru energie optimă și recuperare
- **Noutăți și articole** din lumea atletismului
- **Profiluri de campioni** pentru inspirație

## Caracteristici

- Design modern și responsive
- Optimizat pentru mobil (atleții accesează de pe telefon)
- Navigare intuitivă cu scroll smooth
- Animații subtile pentru experiență plăcută
- Structură modulară pentru actualizări ușoare

## Structura Site-ului

- **Acasă** - Introducere și statistici generale
- **Noutăți** - Articole recente despre atletism
- **Antrenamente** - Programe pe niveluri de experiență
- **Nutriție** - Ghid alimentar pentru atleți
- **Strategie** - Tactici de cursă pentru 800m
- **Profiluri** - Campioni celebri și realizările lor

## Tehnologii

- HTML5
- CSS3 (cu variabile CSS pentru temă)
- JavaScript (vanilla)
- GitHub Pages pentru hosting

## Dezvoltare

Site-ul este construit cu tehnologii web standard, fără framework-uri, pentru performanță maximă și simplitate în mentenanță.

### Adăugare de conținut nou

1. Adaugă articole în secțiunea de noutăți
2. Actualizează programele de antrenament
3. Adaugă profiluri de noi campioni
4. Publică articole de nutriție și strategie
